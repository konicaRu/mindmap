# Editable Mind Map

Быстрая редактируемая mind map на чистом ванильном JS (нативные ES-модули, без сборки).
Сделана по образцу [context-engineering-mindmap](https://safreliy.github.io/context-engineering-mindmap/),
но, в отличие от него, карту можно **редактировать прямо в интерфейсе** и сохранять.

## Запуск

Открыть `index.html` в браузере (двойной клик) или раздать как статику:

```bash
python -m http.server 8099 --bind 127.0.0.1
# затем открыть http://127.0.0.1:8099/index.html
```

Сборка не нужна — это просто статические файлы.

## Возможности

- Радиальная карта: zoom (колесо), pan (перетаскивание фона), drag узлов (позиция сохраняется).
- Режим **«Редактирование»**: правка текстовых полей, добавление/удаление узлов
  (удаление — с подтверждением), смена родителя через выпадающий список.
- Фильтр по тегам (чипы сверху) и текстовый поиск.
- Хранение: **автосохранение в localStorage** + **Экспорт/Импорт JSON**.

## Тесты

```bash
node --test
```

Покрывают чистую логику: модель дерева (`src/model.js`), раскладку (`src/layout.js`),
сериализацию (`src/storage.js`). DOM/рендер проверяются вручную в браузере.

## Структура

| Файл | Ответственность |
|---|---|
| `index.html`, `styles.css` | Оболочка и стили |
| `data.js` | Стартовая (пустая) карта: один корневой узел |
| `src/model.js` | Операции с деревом (add/delete/reparent/setPosition/updateFields) |
| `src/layout.js` | Радиальная раскладка |
| `src/storage.js` | Сериализация + localStorage + экспорт/импорт |
| `src/render.js` | Рендер узлов, связей, дока деталей |
| `src/editor.js` | Форма редактирования узла |
| `src/interactions.js` | Zoom / pan / drag |
| `src/app.js` | Состояние и связка модулей |

## Модель данных

Вся карта — один JSON-объект (дерево). Узел:

```js
{ id, title, summary, why, example, insights[], tags[], sources[{title,url}], position{dx,dy}, children[] }
```

Экспорт/импорт — сериализация этого дерева. `position` хранит ручное смещение узла
поверх автоматической радиальной раскладки.
